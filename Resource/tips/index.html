• TIPS插件,用于操作信息提醒。具体调用方式，请查看API。

示例演示

1.简单demo

	
								$('#tips1').click(function() {
									$.hai_tips({
										title : '温馨提醒',
										show: true, 
										content : ' 团队将致力将用户体验做到极致！ '
								       });
							        });

2.传入页面data

团队将致力将用户体验做到极致！

	
								$('#tips2').click(function() {
										$.hai_tips({ 
											content : document.getElementById('data'),
											opacity : false
										});
									});

3.手动设置高度宽度

	
							$('#tips3').click(function() {
										$.hai_tips({
											title : '温馨提醒',
											content : ' 团队将致力于把租车用户体验做到极致',
											width : 600,
											height : 100
										});
									});

4.有 确定 取消 按钮

	 
							$('#tips4').click(function() {
										$.hai_tips({
											title : '温馨提醒',
											content : ' 团队将致力于把租车用户体验做到极致',
											ok : function() {
												alert("你点了确认");
											},
											cancel : function() {
												alert("你点了取消");
											}
										});
									});

5.提示信息框,3秒后关闭

	
								
						$('#tips5').click(function() {
									$.hai_tips({
										content : ' 团队将致力将用户体验做到极致！',
										type : 'tips'
									});
								});

TIPS API

名称 类型 默认值 描述
title String "温馨提醒" 标题的内容
width Number auto 设置消息内容的宽度。一般不用设置，自适应。
height Number auto 设置消息内容的高度，一般不用设置，自适应。
opacity Number 0.75 遮罩层的透明度，false为不显示遮罩
show Boolean true 是否显示弹出层
content String/HTMLElement null 如果传入的是HTMLElement类型，缓存在页面的数据，方可显示。
overlayClose Boolean true 点击遮罩层是否可以关闭
overlayColor String #4F4F4F 遮罩层的颜色
type String null 消息类型：tips
ok function null 确定按钮回调函数，函数如果返回false将阻止弹出层关闭
cancel function/Boolean true 取消按钮回调函数，如果为true，调用默认关闭事件，函数如果返回false将阻止弹出层关闭
time Number true 多少秒后关闭弹出层
